## Getting Started

Deployed : https://dot-pokemon.netlify.app/

Clone App

```bash
git clone https://github.com/firmansyf/dot-pokemon-app.git
```

Install App 

```bash
npm install or yarn install
```

Running Project

```bash
npm run dev or yarn run dev
```

Feature

- Login
- Search
- Sort
- Filter
- Detail
- Logout
